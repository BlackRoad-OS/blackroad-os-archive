## Quick Setup

1. Copy the configuration files to your device
2. Configure SSH keys for secure access
3. Set up repository access tokens
4. Test connectivity to BlackRoad infrastructure

## Security Notes

- Use SSH key authentication (not passwords)
- Store credentials in device keychain
- Enable biometric unlock where available
- Rotate access tokens regularly
